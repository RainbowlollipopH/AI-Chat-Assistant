## AI Chat Assistant
An AI chatbot project with conversation history management.
這是一個利用 AI API 建立的聊天機器人專案：
將語言模型應用於對話系統，並加入歷史紀錄管理，使模型能根據過去對話維持上下文。

## Features
- AI 對話功能
- 保存對話歷史紀錄
- 整合工具功能

## Development
使用 Python 開發。

主要套件與工具：
- AI API
- peas_agent_tools

## Project Motivation
希望透過實作了解 AI 應用、API 串接以及程式架構設計。

## Notes
API key 使用獨立設定檔管理，並透過 .gitignore 避免公開。

## Learning Experience
透過本專案，我學習了 AI API 串接、對話歷史管理，以及如何使用設定檔分離敏感資訊，
避免將重要資料公開於程式碼中。
